# افزونه KarasuWooPannel

افزونه **KarasuWooPannel** یک پنل مدیریت فروشگاه کاملاً مستقل، راست‌چین (RTL) و مدرن مبتنی بر TailwindCSS برای مدیریت حرفه‌ای ووکامرس است که دسترسی‌های مدیر فروشگاه را بدون نیاز به ورود به بخش سنتی مدیریت وردپرس (`/wp-admin`) فراهم می‌کند.

---

## ویژگی‌های اصلی افزونه

1. **پنل مدیریت مستقل و مدرن:**
   - رابط کاربری کاملاً واکنش‌گرا و راست‌چین شده با TailwindCSS.
   - ورود اختصاصی و مدیریت نشست‌ها جدا از بخش پیش‌فرض وردپرس.
   - حفاظت در برابر حملات Brute Force (محدودیت تعداد دفعات ورود ناموفق).

2. **مدیریت محصولات متغیر و ساده:**
   - ویرایش و ایجاد محصول ساده و متغیر.
   - مدیریت ویژگی‌ها (Attributes) و ایجاد خودکار تنوع‌های محصول (Variations) به روش ضرب دکارتی.
   - آپلود آسان تصویر شاخص محصول و تصاویر تنوع‌ها.

3. **مدیریت دسته‌بندی‌ها:**
   - نمایش درختی سلسله‌مراتبی دسته‌ها با رعایت عمق والد/فرزند.
   - امکان افزودن و حذف دسته‌بندی‌ها به صورت کاملاً زنده (AJAX).

4. **مدیریت کدهای تخفیف (Coupons):**
   - ایجاد، ویرایش و حذف کوپن‌های تخفیف استاندارد ووکامرس.
   - ژنراتور خودکار تولید کدهای تخفیف تصادفی.
   - انتخاب تاریخ انقضا با تاریخ شمسی و تبدیل خودکار آن به میلادی برای ذخیره‌سازی در دیتابیس ووکامرس.

5. **گزارش‌های پیشرفته فروش با تاریخ شمسی:**
   - محاسبه و ترسیم نمودارهای خطی روند فروش بر اساس دوره‌های زمانی شمسی با استفاده از کتابخانه **Chart.js**.
   - خروجی فایل گزارش‌های فروش به فرمت CSV سازگار با Microsoft Excel (با پشتیبانی از UTF-8 BOM).
   - کش کردن خروجی گزارش‌ها با استفاده از Transientهای وردپرس جهت افزایش سرعت و کارایی دیتابیس.

6. **سیستم اطلاع‌رسانی پیامکی (سامانه ملی‌پیامک):**
   - ارسال خودکار پیامک در زمان ثبت سفارش، تغییر وضعیت سفارش به ۷ حالت مختلف برای خریدار.
   - ارسال پیامک‌های هشدار مدیریت برای ثبت سفارش جدید و کاهش موجودی انبار.
   - سیستم لاگ اختصاصی پیامک‌های ارسالی با ثبت جزئیات وضعیت ارسال.
   - ویرایش و شخصی‌سازی کامل قالب‌های متنی پیامک در پنل ادمین وردپرس با قابلیت استفاده از متغیرهای پویا.

7. **سازگاری با المنتور:**
   - ویجت اختصاصی دکمه ورود در المنتور با تنظیمات کامل طراحی و استایل.
   - دسته‌بندی مجزا در پنل المنتور زیر بخش ووکامرس.

8. **سیستم بروزرسانی خودکار گیت‌هاب:**
   - بررسی خودکار نسخه‌های جدید منتشر شده در مخزن گیت‌هاب و امکان آپدیت مستقیم از بخش مدیریت افزونه‌های وردپرس.

---

## جداول اختصاصی دیتابیس

این افزونه از ۳ جدول سفارشی و بهینه‌سازی شده در دیتابیس استفاده می‌کند:
- `wp_wsm_sms_log`: ثبت و نگهداری لاگ‌های پیامک ارسالی (کد وضعیت، متن پیام، تاریخ ارسال شمسی، شماره گیرنده).
- `wp_wsm_sessions`: مدیریت نشست‌های اختصاصی ورود مدیر فروشگاه.
- `wp_wsm_login_attempts`: ردیابی تلاش‌های ورود ناموفق برای جلوگیری از حملات امنیتی.

## نحوه نصب و پیکربندی

1. فایل زیپ افزونه را در مسیر `wp-content/plugins` آپلود و استخراج کنید.
2. افزونه را از بخش مدیریت افزونه‌های وردپرس فعال کنید. (پیش‌نیاز: افزونه ووکامرس فعال باشد).
3. به منوی **ووکامرس > KarasuWooPannel** در مدیریت وردپرس مراجعه کرده و پیکربندی‌های لازم (مانند اطلاعات ملی‌پیامک و آدرس پنل) را انجام دهید.
4. با مراجعه به آدرس اختصاصی تعیین شده برای پنل, وارد محیط مدیریت اختصاصی شوید.

---

## توسعه و فرآیند بروزرسانی خودکار (GitHub Release)

این پروژه مجهز به گردش کار GitHub Actions است که انتشار و بسته‌بندی افزونه را خودکار می‌کند:
1. فایل تغییرات خود را ذخیره کنید و نسخه جدید را در هدر فایل `karasu-woo-pannel.php` و فایل `class-wsm-github-updater.php` تغییر دهید.
2. یک تگ نسخه به صورت `v1.0.1` ایجاد کرده و به گیت‌هاب پوش کنید:
   ```bash
   git tag v1.0.1
   git push origin v1.0.1
   ```
3. ابزار GitHub Actions به صورت خودکار پکیج زیپ افزونه را ساخته و آن را به بخش Release گیت‌هاب شما متصل می‌کند.
4. وردپرس در سایت‌های مقصد بلافاصله نسخه جدید را شناسایی کرده و گزینه بروزرسانی را به مدیر سایت نمایش می‌دهد.
